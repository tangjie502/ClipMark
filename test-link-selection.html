📋 使用说明

  1. 确保已安装并启用 MarkSnip 浏览器扩展
  2. 右键点击页面，选择 "📎 选择页面链接进行批量处理"
  3. 或者点击扩展弹窗中的 "batch" 按钮，然后点击 "📎 从页面选择链接"
  4. 页面将进入链接选择模式，所有可选链接会被蓝色虚线框标记
  5. 按住 Command (Mac) 或 Ctrl (Windows/Linux) + 鼠标左键点击链接进行选择
  6. 选中的链接会变成绿色高亮，并显示 ✓ 标识
  7. 右上角浮动面板实时显示已选择的链接数量
  8. 选择完毕后点击 "选择完毕" 按钮
  9. 扩展会自动打开批量处理界面，并填充选中的链接
⚠️ 注意： 如果功能不工作，请检查浏览器控制台是否有错误信息，并确保扩展已正确加载。
